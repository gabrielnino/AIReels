### **Paso 2: Crear adaptador de qwen-poc output**

```python
def adapt_qwen_to_upload(final_assets: dict) -> VideoMetadata:
    """Convierte output de qwen-poc a formato de uploader"""
    return VideoMetadata(
        caption=final_assets.get("caption", ""),
        hashtags=final_assets.get("hashtags", []),
        # Mapear otros campos si es necesario
    )
```

### **Paso 3: Implementar bridge**
```python
class PipelineBridge:
    def __init__(self, uploader: InstagramUploader):
        self.uploader = uploader

    def process_and_upload(self, final_assets: dict) -> UploadResult:
        metadata = adapt_qwen_to_upload(final_assets)
        return self.uploader.upload_video(
            video_path=final_assets["final_video_path"],
            metadata=metadata
        )
```
